# 🧪 第一部分：基础 API 调用（Qwen）

> **目标：学会如何从代码中调用大语言模型（LLM）——这是构建任何 Agent 的第一步。**

---

## 🔍 为什么从这里开始？

在构建智能 Agent 之前，你必须先掌握：

- 如何获取 LLM 的访问凭证（API Key）
- 如何通过代码向模型发送请求
- 如何接收并解析模型的回复

本部分以 **通义千问（Qwen）** 为例，带你完成从 **零到第一次成功调用** 的全过程。

## 💡 关键知识点

✅ **OpenAI 兼容接口**  
阿里云百炼支持标准 OpenAI API 格式，这意味着你写的代码未来可以**无缝切换**到 OpenAI、Moonshot、Doubao 等其他平台——这是构建多模型 Agent 的基石。

✅ **模型选择意识**  
你将亲身体验 `qwen-turbo`（快而便宜） vs `qwen-max`（强而慢）的区别，为后续 Agent 的性能与成本权衡打下基础。
